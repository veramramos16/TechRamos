# Sentinel-Stream: Automated Threat Intelligence Engine

## 🛡️ Project Overview
Sentinel-Stream is an automated security tool designed to monitor system logs in real-time, identify suspicious activity, and enrich data using external Threat Intelligence APIs.

## 🔍 The Problem
Security analysts are often overwhelmed by "log fatigue." Sentinel-Stream filters the noise by automatically checking IP reputations and only alerting on high-risk indicators.

## 🏗️ Architecture
1. **Ingest:** Tails local system logs (e.g., SSH, Web Server).
2. **Analyze:** Regex-based extraction of IPs and timestamps.
3. **Enrich:** Cross-references IPs with VirusTotal and AlienVault.
4. **Alert:** Sends high-severity alerts to a secure dashboard/webhook.

## 🛠️ Tech Stack
- **Language:** Python 3.12
- **Tools:** Regex, Requests, Dotenv
- **Security:** Bandit (Static Analysis), Secret Masking
